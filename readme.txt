Never leave your website in the dark again, checks to make sure you are not blocking out search engines.

== Description ==

By enabling Inigo SEO Check, your wordpress website will notify you in the event of ‘Discourage search engines from indexing this site’ being switched on in the general settings. 

Once the plugin is enabled it will sit quietly monitoring, once it finds that the setting has been enabled it will show a notice inside wp-admin and also email the admin contact to notify them.

This will help you never leave a website in the dark from Google, by accident, again!

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy

== Frequently Asked Questions ==
